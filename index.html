Setting Up Github

    • Signup For An Account and Go To Your Profile
      Click "New in the top left corner to create a repository"
      Follow the steps until you get to this page.

    • Adding Your Files To Github

    Note: To create your website, you must have a index.html file ready. This file is your main page that GitHub looks for.

    • Now find your project folder, right click on it, and click Open With Git Bash.
      Type git init
      This initializes your file with git, allowing you to use it with GitHub

    We need to setup something right now. Go to your profile settings by clicking the icon in the top right corner of your screen, and clicking settings. Now click developer setting, and click on "Personal Access Tokens." Click on what you want this token to do, and click "Generate New Token." Copy this token and save it for later.

    Now type "git remote /"Your Username/Your Repo Name"

      After this, type "git add ."
      Now type git commit -m "Create page."

    Git now wants to connect with github.

      Type git config --global user.email "Type your Github email address in here"
        Type git config --global user.name "Type your username here"
        If a box asking you for your Github username appears, type it. It will then ask you for your password. Type the token instead.
        Now type git push -u origin master

      Reload the repository page, and see if you can find your index.html file. You may have to wait a few minutes.

    • Publishing Your Site

    Unless you have a paid account, you must have your repository public. You can change this in Settings, Options, Change Repository Visibility, located at the bottom, and set to public.

    • Go to your Github Repository Page, open Settings, Pages.
      Set your Source to master.
      You can now publish your site.

    • Editing Your Site

    • Open git bash by right-clicking on your project folder and clicking,"Open With Git Bash".
      Type "git add ."
      Type git commit -m "Name Of Change"

    Note: Don't type "Name Of Change". Type what you have changed, this helps you to understand what each change does.

      Now type "git push origin master"

    Note: Atom automatically links with your Git Bash. You can press push at the bottom after typing your commit message, and skip token authentication.
